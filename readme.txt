== Description ==

Each day, CHQ outputs one of 365 quotes selected from the 7-volume Cosmic History Chronicles at [__Lawoftime.org__](http://Lawoftime.org/).

Cosmic History Quotes (CHQ) is a collaboration between Juryt Abma and Anthony Fogleman.

Usage: Install plugin, and insert this shortlink into a text widget for your sidebar or directly into a post with sortcode `[cosmic-quotes]`

== Installation ==

= Automatic Method =

1. Use WordPress `Dashboard -> Plugins -> Add New`
2. Search: `Cosmic History Quotes`
3. Click `install now`
4. Activate the plugin `(in Plugins menu)`
5. Insert the shortcode `[cosmic-quotes]` into a page or text widget

= Manual Method =

1. Download the zip file from `AnthonyFogleman.com`
2. Expand contents of zip file.  It will create a directory: `cosmic-history-quotes`
3. Upload contents of zip file to your plugin directory: `/wp-content/plugins/`
4. Activate plugin
5. Insert the shortcode `[cosmic-quotes]` into a page or text widget

= Once installed, how to make it work: =

1. Copy the shortcode: `[cosmic-quotes]` into any page or post ... or
2. Drag a text widget into your sidebar then paste the shortcode `[cosmic-quotes]` into the text widget 
3. Optional settings are at `Dashboard -> Settings -> CHQ Settings`

== Frequently Asked Questions ==

= Can I change the default language =

This version should provide proper localization for en, nl, es until other languages are added.

= Can I place this in any page or post? =

YES! After installing, copy and paste the following code: `[cosmic-quotes]`

= What can I modify about the plugin =

You can navigate to the settings menu, which will be `CHQ Settings` to see what's there, and what can be modified.

You can modify items such as:

* Starting Language
* Font Color
* Background Color
* Width of div
* Font size
* Line height
* Show ads
* Header info position
